# Bellanca_UnityPlateau

Dans cette version du projet, le personnage en question est l'homme des caverrnes (CaveNain) pas le monstre qui fait la toupie

Il peut marcher, courir (Shift) et marcher lentement (C).
L'animation de saut en comprend 3 : Jumping, Falling et Landing
pour quelquonque raison, le perso n'accepte de sauter que lorsqu'il est en mouvement, et rarement quand il est immobile.

Il y a une animation d'ennui où le perso est censé danser, mais pour quelquonque raison elle ne se joue pas correctement, faisant qu'il lève juste les bras en l'air.

Je demande au directeur artistique du projet quel type d'Animator le moteur utilise, je lui demande aussi si le rigging est automatique ou s'il faut le faire manuellement. Ensuite je demande aux programmeur des animations
quel type de technique est employée (State Machine, Blend Tree, Root Motion,Kinematcis ou Procédurales).
